
# Part 2 — Commands You Must Know

| Command | Purpose |
|---|---|
| `show running-config` | Show active configuration |
| `show startup-config` | Show startup configuration |
| `show version` | Device information, IOS, uptime |
| `show ip interface brief` | Quick IP and interface status check |
| `show interfaces status` | Check switch port status |
| `show mac address-table` | Show MAC address table |
| `show running-config interface fa0/1` | Show interface configuration |
| `write memory` | Save configuration |
| `copy running-config startup-config` | Alternative save command |
| `hostname SW1` | Configure device hostname |
| `enable secret PASSWORD` | Secure privileged EXEC mode |
| `description UPLINK_TO_R1` | Add interface description |
| `shutdown` | Disable interface |
| `no shutdown` | Enable interface |
